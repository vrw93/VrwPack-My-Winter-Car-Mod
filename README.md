# Vrw Pack
A Mod Pack For My Winter Car

## Description

A Simple Mod To Add Clock & Unlimited Fuel Into My Winter Car

---
## Feature
- Unlimited Fuel (Gifu, Kekmet, Sorbett)
- Clock In Overlay(24-H/12-H)
- Debugging
---

## How To Install
1. Download [MSCLoader (For My Winter Car)](https://www.nexusmods.com/mywintercar/mods/3) And Install It
2. Download The Mod [Here](https://github.com/vrw93/VrwPack-My-Winter-Car-Mod/releases)
3. Locate My Winter Car Dir Location usually at `~/.local/share/Steam/Amistech/My Winter Car`
4. Go Inside Mods Folder `/Mods`
5. Put The Mod There
6. And Start The Game
